Features

何ができるのか

🔍
Web リサーチ自動化
テーマに沿った最新情報をWebから自動収集。出典URL付きで事実確認済みの記事を生成します。
✍️
本文・見出し・太字まで自動生成
H2/H3見出し、箇条書き、太字対応。note.comのエディターに直接書き込みます。
🖼️
サムネ＆図解を自動生成
HTMLテンプレートからサムネイルと差し込み画像を自動生成。Codex CLIにも対応。
🚀
note.comへ直接下書き投稿
APIキー不要。ブラウザのログインセッションだけで動作。下書き保存後にハッシュタグをターミナルに出力します。
⚡
初回セットアップも自動
npm install や Playwright インストールはClaude Codeが自動実行。手動操作は最初のログインのみ。
🔒
機密情報はGitに含まれない
.gitignore 設定済みで、ログインセッションファイル・.envは自動で除外。安全に配布できます。
Setup

セットアップ手順

1
リポジトリをクローン
Claude Codeのチャット欄に以下をそのまま貼り付けてEnterを押してください。
! git clone https://github.com/seadstamusho/note-meteor.git
2
スキルを起動する
チャット欄でスキルを呼び出すと、npm install・Playwright インストールを自動で実行します。
note-meteorスキルを使って「〇〇について」という記事を書いて
3
note.com にログイン（初回のみ）
スキルの案内に従い、チャット欄に以下を入力してEnterを押すとブラウザが開きます。いつも通りログインして閉じてください。
! npm run login
4
あとはおまかせ
リサーチ → 執筆 → 画像生成 → note.com下書き投稿まで自動で進みます。完了後にハッシュタグが表示されるので、コピペして公開設定から貼り付けてください。
FAQ

よくある質問

APIキーは必要ですか？
不要です。note.comへのアクセスはブラウザのログインセッション（Cookie）を使います。Geminiも使いません。
ログインは毎回必要ですか？
初回のみです。セッション情報がローカルに保存されるため、2回目以降は自動的にログイン済み状態で投稿できます。
ハッシュタグが自動で設定されないのはなぜ？
noteのUI変更によりPlaywrightでの自動入力が困難になりました。代わりに、完了後にターミナルへハッシュタグ一覧を出力します。コピペで一撃です。
「！」（感嘆符）は何のために使うの？
Claude Codeのチャット欄で「!」をつけるとターミナルコマンドとして実行されます。「! npm run login」と入力することで、別のターミナルを開かずに操作できます。
